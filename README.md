# Greenden

A beautiful and responsive website built with HTML and Tailwind CSS.

## Table of Contents
* [Description](#description)

## Description
Greenden is a modern and responsive website designed to showcase the beauty of nature. The website is built using HTML and Tailwind CSS, making it fast, efficient, and easy to maintain.

## Features
* Responsive design for a seamless user experience on all devices
* Clean and modern layout with a focus on typography and whitespace
* Customizable colors and fonts for easy branding
* Fast and efficient loading times thanks to Tailwind CSS

## Technologies Used
* HTML5
* Tailwind CSS
